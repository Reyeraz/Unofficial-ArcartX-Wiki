# 常用使用技巧
## 控件属性的用法
控件属性和UI Action 中的tick一样, 每秒钟更新一百次, 再加上可以任意自定义属性名, 那么可以做到
- 创建一个自定义属性名, 用于维护一个状态, 比如你需要维护一个百分比的数值, 需要实时使用, 那么可以使用这个属性名, 然后使用 `self.你的属性名` 可以作为一个需要实时更改变量的代称
  示例:
    ```yaml
        控件:
            type: texture
            attribute:
                normal: ~255,255,255
                health: Player.getHealth() / Player.getMaxHealth()
                width: self.health * 100
    ```
- 有一些控件名是无效的, 如果你遇到有一些控件不显示, 请试图更改为其他的控件名
- 控件属性的键值支持方式如下:
    ```yaml
        attribute1: ~纯字符
        attribute2: "字符" # 这个是字符串字面量,在Arcart中会被尝试当作一个变量进行编译
        attribute3: var.字符 # 这个是一个字符串变量,当然也可以是其他类型的变量
        attribute4: var.数值 / var.另外一个数值 # 这个结果会直接进行每秒一百次的运算
        # 当然你也可以使用多种复合函数进行运算, 但是这样会使得代码很难看, 使用创建自定义属性名进行简化
    ```
- 使用 `self.action['一个动作名']`, 来调用自身动作或者你的自定义动作, 当然,如果你需要操控其他控件,`self`也可以替换成`val.一个控件名` 或是 `self.parent()`
    ```yaml
        控件:
            type: texture
            action:
                open: |-
                    // 一堆东西
                    self.action['other']
                    //  执行到该语句时会调用other的行为发生器
                other: |-
                    // 一堆东西
    ```
