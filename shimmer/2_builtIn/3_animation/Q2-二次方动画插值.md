# Q2-二次方动画插值

二次方动画插值对象及其函数

人是必须要被超越的东西

Q2是一个基于二次方函数(t²)的插值对象，可以创建平滑的加速或减速效果。相比于更高次幂的插值器，它提供了较为柔和的动画效果。

## 实例化方式

### `Q2`含参实例化

  * **描述** ：创建一个二次方插值实例。
  * **参数** ：
    * `start`：数值类型，起始值
    * `end`：数值类型，结束值
    * `type`：整数类型，动画类型（0=easeIn, 1=easeOut, 2=easeInOut）
    * `transferTime`：长整型，过渡时间（毫秒）
  * **调用示例** ：

    
    Q2(0, 100, 0, 1000) // 1秒内从0到100，使用easeIn效果
### 动画类型说明

  * **easeIn(0)** ：渐入效果，先慢后快
  * **easeOut(1)** ：渐出效果，先快后慢
  * **easeInOut(2)** ：渐入渐出效果，两端慢中间快

### 默认参数

当不提供完整参数时，使用以下默认值：

  * **动画类型** : 0 (easeIn)
  * **过渡时间** : 1000毫秒

## 函数

### `get()`

  * 获取当前插值计算的结果

  * **参数** ：无
  * **返回值** ：数值类型
  * **说明** ：根据当前时间计算二次方插值结果
  * **调用示例** ：

    
    Q2(0, 100, 0, 1000).get()
### `reset()`

  * 重置插值器状态

  * **参数** ：无
  * **返回值** ：无
  * **说明** ：重置计时器并清除反向状态，重新开始动画
  * **调用示例** ：

    
    Q2(0, 100, 0, 1000).reset()
### `reverse()`

  * 反转动画方向

  * **参数** ：无
  * **返回值** ：无
  * **说明** ：将动画方向反转，从当前位置开始向起始值过渡
  * **调用示例** ：

    
    Q2(0, 100, 0, 1000).reverse()
## 类型转换

Q2对象支持以下类型转换：

  * **数值** ：返回当前插值计算的结果
  * **布尔值** ：当前值大于0时返回true，否则返回false
  * **字符串** ：返回当前插值计算结果的字符串表示

Q2对象支持数学运算，可以直接参与计算表达式。
