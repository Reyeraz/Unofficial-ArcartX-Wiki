# Bounce-弹跳动画插值

弹跳动画插值对象及其函数

人类唯有生长在爱中，才得以创造出新的事物

Bounce是一个模拟物体弹跳效果的插值对象。通过设置弹跳次数和衰减系数，可以创建真实的弹跳动画效果。

## 实例化方式

### `Bounce`含参实例化

  * **描述** ：创建一个弹跳插值实例。
  * **参数** ：
    * `start`：数值类型，起始值
    * `end`：数值类型，结束值
    * `bounces`：整数类型，弹跳次数
    * `decay`：数值类型，衰减系数(0-1)
    * `transferTime`：长整型，过渡时间（毫秒）
  * **调用示例** ：

    
    Bounce(0, 100, 3, 0.5, 1000) // 1秒内从0到100，弹跳3次，每次衰减50%
### 默认参数

当不提供完整参数时，使用以下默认值：

  * **弹跳次数** : 3次
  * **衰减系数** : 0.5 (50%衰减)
  * **过渡时间** : 1000毫秒

## 函数

### `get()`

  * 获取当前插值计算的结果

  * **参数** ：无
  * **返回值** ：数值类型
  * **说明** ：根据当前时间计算带弹跳效果的插值结果
  * **调用示例** ：

    
    Bounce(0, 100, 3, 0.5, 1000).get()
### `reset()`

  * 重置插值器状态

  * **参数** ：无
  * **返回值** ：无
  * **说明** ：重置计时器并清除反向状态，重新开始动画
  * **调用示例** ：

    
    Bounce(0, 100, 3, 0.5, 1000).reset()
### `reverse()`

  * 反转动画方向

  * **参数** ：无
  * **返回值** ：无
  * **说明** ：将动画方向反转，从当前位置开始向起始值过渡
  * **调用示例** ：

    
    Bounce(0, 100, 3, 0.5, 1000).reverse()
## 类型转换

Bounce对象支持以下类型转换：

  * **数值** ：返回当前插值计算的结果
  * **布尔值** ：当前值大于0时返回true，否则返回false
  * **字符串** ：返回当前插值计算结果的字符串表示

Bounce对象支持数学运算，可以直接参与计算表达式。
