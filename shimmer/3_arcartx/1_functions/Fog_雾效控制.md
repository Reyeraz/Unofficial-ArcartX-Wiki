# Fog:雾效控制

Fog雾效渲染控制工具集

自由就是不受他人任意干涉的权利

## `Fog.setup(r, g, b, start, end)`

  * 设置雾效参数

  * **参数** ：
    * `r`: 整数类型(红色分量，0-255)
    * `g`: 整数类型(绿色分量，0-255)
    * `b`: 整数类型(蓝色分量，0-255)
    * `start`: 浮点数类型(雾效起始距离)
    * `end`: 浮点数类型(雾效结束距离)
  * **调用示例**

    
    // 设置白色雾效，从5个方块开始，延伸到20个方块
    Fog.setup(255, 255, 255, 5.0, 20.0)
     
    // 设置红色雾效，从2个方块开始，延伸到15个方块
    Fog.setup(255, 0, 0, 2.0, 15.0)
## `Fog.clear()`

  * 清除当前雾效设置

  * **调用示例**

    
    Fog.clear()
注意事项：

  * RGB颜色值范围为0-255
  * 起始距离必须小于结束距离
  * 过大的雾效范围可能影响游戏性能和视觉体验
  * 清除雾效后将恢复到游戏默认的环境效果
