## 2\. 理解代码结构

### 工具集 (Shimmer)

就像一个工具箱，里面装着各种的工具。

    
    Shimmer.println("你好，世界！")  // Shimmer是工具箱的名字
    ↑↑↑↑↑↑↑
### 工具 (println)

工具箱里的具体工具，这里的println是用来显示文字的工具。

    
    // 从工具箱里取出工具
    Shimmer.println("你好，世界！")
            ↑↑↑↑↑↑↑
### 参数 ("你好，世界！")

告诉工具具体要做什么，就像告诉微波炉要加热什么食物。

    
    // 括号里的内容就是参数
    Shimmer.println("你好，世界！")
                   ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑
在后文中，这种调用方式我们**统称为"工具集"调用** ，而括号里面的内容则称为参数。
