## 实体血条

  * 看见这个名字，应该是非常的好理解，生物血条则是将全息渲染在生物头顶，来实现显示生物血量的功能。

## 配置格式

  * 生物血条配置位于 `hologram/entity_heal_bar` 目录下。

    
    example:
      match:           # 匹配的生物名称（如果是默认的，这里填写default即可，如果实体名一个没匹配上就显示被设置为default的血条）
                            # 此外，还有以下类型：AdyeshachOnly | PlayerOnly | OtherEntity（该类型就是不包含前两种）
        - "boss"
      keepTime: 5000   # 显示持续时间(-1为永久)
      fadeTime: 500    # 淡入淡出时间
      hologram: "测试血条"  # 血条模板名这个模板名就是你在data目录里创建的全息模板的文件名，比如“全息示例.yml”这里就是填写“全息示例”
  * 这块也是相当的简单，只需要填写匹配的生物名称，显示持续时间，淡入淡出时间，以及血条模板名即可。
  * 其中，对于生物血条的全息，附带一些额外的变量：

    
    # val.name:生物名称
    # val.health:生物血量
    # val.maxHealth:生物最大血量
