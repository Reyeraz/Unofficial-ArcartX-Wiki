# 基础设定

相机基本设定指南

对待生命你不妨大胆冒险一点，因为好歹你要失去它，何必总陷于一片泥土。

ArcartX提供了全面的自定义相机，包括自定义相机位置、八向移动相机、场景相机以及ALT巡视相机等功能。本文将详细介绍相机系统的配置和使用方法。

## 相机基础设定

相机的默认设置位于 `camera/setting.yml`：

    
    # 相机功能总开关
    enable: true
     
    # 强制第三人称视角
    force: true
     
    # 默认使用的预设ID
    default: "idle"
  * 应该是非常简单，只有三个选项，如果总开关不开启或者说不存在默认预设，相机功能是不会正常加载的。
  * 至于预设是什么东西，我们看下一章。
  * 此外，ArcartX的相机功能内置了ALT辅助巡视相机，按下ALT即可自由巡视，当然，这个功能关不掉，我懒得整个开关了。
