# 数据类型

认识Shimmer中的基本数据类型

理论不能解决的问题，实践或许能解决

在编程中，不同的信息需要用不同的方式来表示。就像现实生活中，我们用数字表示年龄，用文字表示姓名一样。

## 基本数据类型

### 数字类型

不带小数点的数字:

    
    age = 18
    count = 100
    neg = -10
带小数点的数字:

    
    height = 1.75
    weight = 65.5
    
    totalScore = 12.5 + 24.8
    averageScore = (85 + 92 + 78) / 3
### 字符串类型

📝

普通字符串

使用单引号的文本:

    
    name = '小明'
    message = '你好啊!'
✨

增强字符串

使用双引号，可以嵌入表达式:

    
    age = 18
    info = "我今年{age}岁"
📄

多行字符串

使用三个双引号，可以写多行:

    
    taiyouyale = """
    春眠不觉晓，
    处处闻啼鸟。
    夜来风雨声，
    花落知多少。
    """
### 布尔值

布尔值只有两种状态：true（真）和false（假），用于表示"是"或"否"

    
    rain = true
    hungry = false
     
    if(hungry){
        Shimmer.println('我要恰饭!')
    }
## 复合数据类型

### 列表

列表就像一个有序的容器，可以存放多个数据:

    
    fruit = ['苹果', '香蕉', '橙子']
    fraction = [98, 86, 92, 77]
    
    // 获取第一个水果(注意：序号从0开始)
    first = fruit[0]  // 结果是'苹果'
     
    // 修改列表内容
    fruit[1] = '草莓'     // 把'香蕉'换成'草莓'
     
    // 拼接另一个列表 或者插入一个新元素
    fruit += ['蓝莓', '西瓜']
    fruit += '一时间想不起来有啥水果了，下次补上'
     
    // 删除某个下标的元素 或者删除和某个列表的重复值
    fruit -= ['草莓','苹果']
    fruit -= 1 // 删除下标位于1的元素
### 字典

字典像是一本查找手册，通过名字(键)来查找对应的内容(值)

    
    // 创建一个学生信息字典
    student = {
        '姓名': '小明',
        '年龄': 15,
        '爱好': ['足球', '画画']
    }
     
    // 使用字典
    Shimmer.println(student['姓名'])    // 显示'小明'
    Shimmer.println(student['爱好'][0]) // 显示'足球'
     
    // 字典合并
    student += {'身高': 145}
     
    // 删除和另一个字典的重复值
    student -= {'身高': 145}
    // 删除列表中包含的键值
    student -= ['年龄','爱好']
    // 删除某个键
    student -= '身高'
