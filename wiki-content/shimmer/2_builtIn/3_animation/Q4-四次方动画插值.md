# Q4-四次方动画插值
四次方动画插值对象及其函数
人可以控制行为，却不能约束感情，因为感情是变化无常的.
Q4是一个基于四次方函数(t⁴)的插值对象。它提供了比Q2和Q3更加剧烈的加速或减速效果，适合需要突出动画戏剧性的场景。
## 实例化方式
### `Q4`含参实例化
  * **描述** ：创建一个四次方插值实例。
  * **参数** ：
    * `start`：数值类型，起始值
    * `end`：数值类型，结束值
    * `type`：整数类型，动画类型（0=easeIn, 1=easeOut, 2=easeInOut）
    * `transferTime`：长整型，过渡时间（毫秒）
  * **调用示例** ：


    
    Q4(0, 100, 0, 1000) // 1秒内从0到100，使用easeIn效果
### 动画类型说明
  * **easeIn(0)** ：极强的渐入效果，初期几乎静止
  * **easeOut(1)** ：极强的渐出效果，末期几乎静止
  * **easeInOut(2)** ：极强的渐入渐出效果，两端变化极其缓慢


### 默认参数
当不提供完整参数时，使用以下默认值：
  * **动画类型** : 0 (easeIn)
  * **过渡时间** : 1000毫秒


## 函数
### `get()`
  * 获取当前插值计算的结果


  * **参数** ：无
  * **返回值** ：数值类型
  * **说明** ：根据当前时间计算四次方插值结果
  * **调用示例** ：


    
    Q4(0, 100, 0, 1000).get()
### `reset()`
  * 重置插值器状态


  * **参数** ：无
  * **返回值** ：无
  * **说明** ：重置计时器并清除反向状态，重新开始动画
  * **调用示例** ：


    
    Q4(0, 100, 0, 1000).reset()
### `reverse()`
  * 反转动画方向


  * **参数** ：无
  * **返回值** ：无
  * **说明** ：将动画方向反转，从当前位置开始向起始值过渡
  * **调用示例** ：


    
    Q4(0, 100, 0, 1000).reverse()
## 类型转换
Q4对象支持以下类型转换：
  * **数值** ：返回当前插值计算的结果
  * **布尔值** ：当前值大于0时返回true，否则返回false
  * **字符串** ：返回当前插值计算结果的字符串表示


Q4对象支持数学运算，可以直接参与计算表达式。
