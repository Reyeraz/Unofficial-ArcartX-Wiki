## MythicMobs额外配置
ArcartX提供了一系列MythicMobs的拓展配置：
    
    test:
      Type: ZOMBIE
      Display: '&a示例'
      Health: 1
      Model: "test"  # 可直接在此设置模型id而不用技能设置
      Scale: 0.5 # 可直接在此设置模型缩放而不用技能设置
      HideName: true # 是否显示名称，比如您的这个生物是一个陷阱类型的，那可能需要隐藏名称标签
      HideHitBox: true # 是否隐藏碰撞箱，用于一些遁地或者模型隐身的怪物隐藏碰撞提及防止玩家透视
