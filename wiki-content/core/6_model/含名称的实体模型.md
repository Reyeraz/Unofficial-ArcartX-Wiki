# 含名称的实体模型
通过实体名称渲染实体模型
我们的眼睛就是我们的监狱，我们的眼光所到之处就是我们监狱的围墙。
## 根据名称渲染实体模型
  * 以名称设置实体模型的配置非常简单，配置位于ArcartX根目录下的`entity_model`目录中，您可以创建多个实体模型配置。
  * 配置格式说明


    
    example: # 这个名字无所谓，别重复就行
      matchName: "123456" # 匹配名，无需颜色符号
      modelID: "test" # 模型ID，上一章提到过获取方式
      scale: 1 # 模型缩放
      hideNameTag: false # 是否隐藏名称标签 让一些装饰品实体不显示名字
  * 配置好了之后将实体命名为matchName的名字，就可以看到生物模型了。
