# 世界全息
世界全息部署指南
人可以控制行为，却不能约束感情，因为感情是变化无常的。
## 世界全息
  * 世界全息便是将我们创建好的全息模板显示在世界上，比如一些装饰性的全息、或者是一些全息排行榜等动态的全息显示。
  * 世界全息配置位于 `hologram/location` 目录下。


## 配置文件
    
    example:
      world: "world"    # 显示世界
      x: -966          # X坐标
      y: 110           # Y坐标
      z: 186           # Z坐标
      yaw: 45          # 水平旋转角度
      pitch: 45        # 垂直倾斜角度
      follow: false    # 是否跟随玩家视角
      hologram: "全息示例"  # 全息模板名，这个模板名就是你在data目录里创建的全息模板的文件名，比如“全息示例.yml”这里就是填写“全息示例”
  * 上面都有注释，应该非常易于理解，这里多提一嘴，在上一个章节我们提到了“实例”。
  * 比如这里我们通过世界坐标显示的全息，就是一个全息实例，和模板不同，每个实例是独立的，其内部的脚本上下文变量都是独立于实例之中的。
  * 如果听起来有点听不懂，没关系，不懂也没啥事。您能按照流程把全息显示出来就行。
